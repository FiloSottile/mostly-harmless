Superfish removal guide

So you got a YES on the Badfish test: https://filippo.io/Badfish/

Let's work together to fix that. Important: don't do this if you don't feel comfortable following the instructions, ask for help, these are administrative operations.

Step 1. Uninstall the software

  1. Open the Windows Start menu or Start screen and search for "Uninstall a program". Launch it.
  2. Right-click on "Superfish Inc VisualDiscovery" and select "Uninstall," then enter your administrator password.

Step 2. Remove the certificate from Windows

  1. Open the Windows Start menu or Start screen and search for "certmgr.msc". Launch it.
  2. Click on "Trusted Root Certification Authorities" and open Certificates.
  3. Look for any certificates that include Superfish Inc, and right-click to delete them


credit: Chris Palmer

Step 3. Remove the certificate from Firefox

This might or might not be needed, but check to be sure.

  1. Got to Options/Preferences
  2. Click Advanced, then Certificates
  3. Click View Certificates
  4. Look for Superfish, if it's there click it and then click the "Delete or Distrust" button

Step 4. Restart the browser

Close completely and reopen your browsers. A reboot will do, too.

Step 5. Check again

Load https://filippo.io/Badfish again from both Chrome/IE and Firefox, and make sure you get a No this time.

Note: The test might still be stuck on the old result, once you think you removed Superfish you can also load canibesuperphished.com and check if you get any warnings. If you do, you're fixed.
